# Original-Blog
### 项目介绍
#### 前言
该项目采用前后端分离的架构，后端主要使用 SpringBoot + Spring Data JPA + MySQL，前端基于。
#### 模块划分
项目在功能上分为两部分：
博客网站，即博客的展示页面。
后台管理平台，即对网站内容进行增删改查的平台。
